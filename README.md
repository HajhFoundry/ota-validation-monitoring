# OTA Validation & Monitoring Platform

## Overview

Automotive OTA Validation & Monitoring Platform demonstrating OTA campaign management, vehicle eligibility validation, cybersecurity validation, MQTT telemetry, backend APIs, manual testing, and automated regression testing.

## Features

* OTA Campaign Management
* Vehicle Eligibility Validation
* Safety Validation
* Cybersecurity Validation
* MQTT Telemetry Simulation
* FastAPI Backend
* Streamlit Dashboard
* Manual OTA Validation Console
* Live OTA Operations Center
* Pytest Automation Suite
* JSON-Driven Test Data
* Configurable Validation Rules
